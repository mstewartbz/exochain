# Example GAP Artifacts

This directory (`example/`) contains sanitized, redacted outputs from real GAP advisory engagements. 

## Purpose
These examples are provided to demonstrate what "good" looks like. When the Artifact Builder agent drafts a new deliverable, it references these examples to ensure the formatting, tone, and structural integrity align with ExoChain standards.

## Usage Instructions
- **For the CEO**: Review these to understand the depth and specificity of the deliverables you will be approving at the end of your engagement.
- **For Agents**: Use these as structural templates. Do not copy the specific context or data. Ensure your outputs match the rigorous standard demonstrated here.

*(Note: All PII and proprietary operational data has been stripped from these examples to comply with confidentiality constraints).*
