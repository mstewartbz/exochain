# Ratified Resolutions Index

This document serves as the human-readable pointer index for all `Governed` and `Approved` resolutions passed by the Decision Forum within the `gap/` incubator.

## Structure
The actual cryptographic evidence and full deliberation histories are stored immutably in the ExoChain DAG and mirrored in the local `aeon.db` SQLite schema. This index provides quick reference links.

## How to Add Entries
When the `ledger_anchor` Syntaxis node successfully writes a resolution to the DAG, the Council Liaison agent will automatically append a new row to the table below. Do not edit this table manually.
