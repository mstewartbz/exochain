## Ranking Rule

Select the highest-value slice that can be completed with:

- a failing test first,
- synthetic fixtures,
- no raw sensitive data,
- no public trust claim,
- no EXOCHAIN core modification,
- one local validation gate,
- one deploy verification when runtime or deployment files changed.
