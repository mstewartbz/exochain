# Commander’s Critical Information Requirements (CCIR)

Only CCIRs may interrupt via Slack/SMS. Everything else stays on the daily brief or auto-route.

## Living CCIR list (initial)

1. Constitutional invariant violation detected in production/runtime probe
2. Dual-gate deadlock on irreversible decision past SLA
3. AI-IRB / council deadlock on Strategic/Constitutional class
4. Kill-criteria trip on any Mission Graph node
5. Railway deploy health/ready failure for EXOCHAIN node after promote
6. Push-path integrity failure / `chairman_unreachable`
7. Adjacent surface claiming constitutional trust without adapter evidence
8. Chaos-drill failure (empty-evidence IRB accepted or SMS path broken)

## Non-CCIR (do not push)

- Routine task completions
- Heuristic ExoForge panel results
- Doc-only PRs
- Informational CQI metrics below threshold
